    • Penjelasan HTML

    HTML adalah singkatan dari HyperText Markup Language.
    Ini adalah bahasa markup yang digunakan untuk membuat
    struktur konten pada halaman web. di html terdapat elemen
    elemen yang digunakan untuk mendesain web agar web
    terlihat lebih menarik.

    Lanjut, pertama ada HTML ,HTML apa bila diibaratkan itu
    seperti kerangka dari sebuah web ibarat bila di tubuh manusia
    itu tulangnya, kedua ada css , css bertujuan untuk memberikan
    atribut pada kerangka html, ibarat bila di tubuh manusia itu
    pakaian nya, ketiga ada java script , yang digunakan untuk
    website lebih interaktif, ibarat bila di tubuh manusia itu
    gerakan gerakan tubuhnya.

    Jika kalian ingin mencoba membuat website sendiri bisa
    dengan membuat new text document dengan nama index.html.
    Setelah itu kalian bisa mengetik ! dan lalu di tab akan muncul
    struktur html yang terdiri dari html, head, title, dan body
    jadi kita bisa mengetikkan codingan kita di bagian body.

    Kita mulai dengan cara membuat paragraf cara membuat
    paragraf bisa dengan tag <p> selanjutnya kita bisa membuat
    katakata tebal , biasanya digunakan untuk judul menggunakan
    tag <h1> h1 biasa disebut heading 1 setelah itu ada <h2>
    perbedaannya semakin angka nya besar ketebalan nya semakin
    menurun sampai maks.<h6>

    Setelah kalian sudah membuat kerangka kerangkanya kalian bisa
    menghubungkan htmlnya ke css dengan menggunakan
    <link> src="style.css" baik skarang kalian sudah menghubungkan
    html dn css kalian, jangan lupa membuat file dengan nama style.css.
    Bagaimana kalau java.script 1. kalian buat file nama script.js
    setelah itu kalian buat <script> dan didalamnya kasih src="script.js".
